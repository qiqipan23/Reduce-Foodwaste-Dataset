Machine Learning Solution

Daily sales patterns are not strictly linear: weekday effects interact with holidays, and weather can shift demand in subtle ways. Tree-based models are well-suited to structured forecasting tasks like this.

Random Forest

Strong baseline; robust to noise; provides useful feature importance signals.

LightGBM

Fast gradient boosting; strong accuracy on tabular data; captures complex interactions efficiently.

XGBoost

Highly competitive boosted trees; robust performance; strong benchmark with careful tuning.

Features used

  • Date-derived signals (weekday patterns)
  • Store identifier
  • Holiday indicators (school/state/special days)
  • Weather variables (temperature, sunshine, precipitation)

Evaluation

We compare models using standard forecasting error metrics to measure average accuracy and stability across stores and time periods.

Models: Random Forest · LightGBM · XGBoost
